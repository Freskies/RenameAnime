How does it work?

This program change the name of all files in the given directory (and subdirectories)
according to the following rules:
all the names are picked up from a file called "names.txt" (one name per line)
"names.txt" must be in the same directory of the program
obviously, the number of names must be equal to the number of files in the directory
("names.txt" doesn't count as a file and is not considered)
"names.txt" MUST finish with a blank line
"names.txt" MUST NOT contain any blank line in the beginning or middle of the file
is ILLEGAL to have a folder with a "." in the name
Pay attention to the fact that the program will change the name of all the files in the directory
and subdirectories, so be sure to have a backup of the files you want to keep!
Enjoy!

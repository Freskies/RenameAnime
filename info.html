How does it work?

This program changes the name of all files in the selected folder except "name.txt"
the number of files must be equal to the number of lines in the "name.txt" file
The file must terminal with a black line
the files are renamed: "(counter) - (filename from" name.txt "). (extension)"
THERE MUST NOT BE folders that have "." in their name!
THERE MUST NOT BE any other files that are not to be renamed!
